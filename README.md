# Midjourney Prompt–Embedding Dataset

This repository contains the dataset associated with our COLM 2024 paper:

**Iteratively Prompting Multimodal LLMs to Reproduce Natural and AI-Generated Images**  
Ali Naseh, Katherine Thai, Mohit Iyyer, Amir Houmansadr  

📄 Paper: [PDF](https://people.cs.umass.edu/~amir/papers/2024-COLM-prompt.pdf) | [arXiv](https://arxiv.org/pdf/2404.13784)

This dataset is derived from our study of whether multimodal language models can iteratively infer prompts that reproduce target images generated by text-to-image systems. The paper analyzes the structure of real-world prompts and their relationship to generated images. To support further research in this direction, we release a large-scale collection of anonymized prompts and corresponding image embeddings.
